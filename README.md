## How to contribute?
**This note is written in markdown**.  For equations I have used LaTex Equations. 

1. At first learn this,If you don't know.It's will take just 10 minutes only.

[Learn Latex equations in 5 minutes](https://medium.com/@6unpnp/some-latex-equations-in-markdown-58670c258b45)

[Learn markdown in 5 minutes](https://www.remarq.io/articles/five-minutes-to-markdown-mastery/)

2. Now fork this repository in github.You will need a GitHub account to fork it.

 [**Fork Modern Physics Note**](https://github.com/withlimon/Modern-Physics-Note/fork)
 
3. Now visit you GitHub account account [home page](https://github.com)
4. You will find "Modern Physics Notes" repository there. Edit or extend it ,then pull a request.That's it!
 

> **⚠️Warning:**  During commit add a proper explanation of your commit in "details" fieldWhy you edited or extended the note.
> It's help to prevent bad edit/extension
> After reveiw of your contribution,your contribution will automatically displayed in the [top](#contributers) of this page.And New PDF relase will be generated in release page of this note.

Thank you!
